 YGT Blog App – Proje Dokümantasyonu

Bu repository, **Blog Uygulaması (YGT Blog App)** projesinin **dokümantasyonunu** içermektedir.

Proje toplamda **2 ana uygulamadan** oluşmaktadır:

| Repository | Açıklama |
|-----------|----------|
| **ygt_blog_app_backend** | API, kullanıcı doğrulama, blog CRUD işlemleri |
| **ygt_blog_app_frontend** | Kullanıcı arayüzü ve blogların görüntülenmesi |

---

Projenin Amacı
Kullanıcıların kayıt olabildiği, giriş yapabildiği ve blog yazıları oluşturup düzenleyebildiği bir web uygulaması geliştirmek.

---

Kullanılan Teknolojiler
- Backend: **Ruby on Rails**, JWT Authentication  
- Frontend: **Next.js (React)**  
- Veritabanı: SQLite / PostgreSQL  
- API iletişimi: RESTful API
